Amon2-Setup-Flavor-Akimacho
====

My Amon2 flavor

## Install

To install this module, run the following commands:

```
$ perl Build.PL
$ ./Build
$ ./Build install
```

## Setup

```
$ amon2-setup --flavor=Akimacho MyApp
```
